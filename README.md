# MindGuesser

MindGuesser is a simple and fun number guessing game built in Java. Players try to guess a randomly generated number, receiving hints after each guess, and can replay as many times as they like.

## How to Play

1. Open `src/MindGuesser.java` in your IDE or text editor.
2. Compile and run the program:
   ```bash
   javac src/MindGuesser.java
   java -cp src MindGuesser
